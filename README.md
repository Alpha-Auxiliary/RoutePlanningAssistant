# Slay the Spire 2 - Route Planning Assistant

这是一款为 **Slay the Spire 2 (杀戮尖塔2)** 开发的地图路径规划辅助 Mod。它可以根据您关注的图例类型（如精英怪、商店、休息处等），自动在地图上计算并绘制出最优化的行进路线。

---

## ✨ 主要功能

- **智能路径搜索**：自动计算从当前位置出发，能够经过最多“目标类型”节点的最佳路径。
- **多图例筛选**：支持在地图图例面板上通过 **复选框 (Checkbox)** 同时勾选多个目标类型（例如：同时追求最多的商店和精英怪）。
- **多路径切换**：当存在多条权重相等的最佳路线时，可通过图例面板右下角的 **“Next Route”** 按钮循环切换不同的方案。
- **UI 无缝集成**：
  - 复选框直接集成在原生的图例项右侧。
  - 点击图例图标即可快速切换选中状态。
  - 路径通过游戏原生的地图绘画系统（MapDrawings）实时呈现。

## 🛠️ 安装方法

1.  **环境要求**：
    - 已安装《杀戮尖塔 2》。
    - 确保游戏支持加载 Mod（通常需要将 Mod 文件放入游戏根目录下的 `mods` 文件夹）。

2.  **获取文件**：
    - 编译生成的 `RoutePlanningAssistant.dll`。
    - 项目中的 `RoutePlanningAssistant.json` 配置文件。
    - Godot打包的 `RoutePlanningAssistant.pck` 文件。

3.  **部署**：
    - 将以上三个文件放入：`[你的游戏安装目录]/data_sts2_windows_x86_64/mods`。

## 🎮 使用手册

1.  **打开地图**：进入游戏后的地图界面。
2.  **选择目标**：在屏幕右侧的图例面板中，勾选你想要重点经过的图标（如精英、商店、宝箱）。
3.  **查看结果**：地图上会自动出现一条绿色的引导线，指引你前往包含最多目标点的路径。
4.  **切换路线**：如果发现当前引导线不是你想要的，点击图例面板右下角的 **“Next Route”** 按钮，尝试其他等效的最优路线。

## 🔨 开发与构建

本项目基于 **Godot .NET SDK 4.x** 开发，使用 **Harmony** 进行运行时代码注入。

### 构建步骤：
1.  修改 `RoutePlanningAssistant.csproj` 中的 `<Sts2Dir>` 路径，指向你的游戏安装目录。
2.  在项目根目录下执行：
    ```bash
    dotnet build -c Debug
    ```
3.  项目已配置 `AfterTargets="PostBuildEvent"`，构建成功后会自动将生成的文件复制到游戏的 `mods` 文件夹。
